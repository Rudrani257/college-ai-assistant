# 🎓 College AI Assistant

AI-powered college assistant web application built using Flask, MongoDB, HTML, CSS and JavaScript.

## 🚀 Features
- Student Login System
- Admin Dashboard
- AI Chat Assistant
- College Information Management
- Responsive UI Design

## 🛠 Technologies Used
- Python
- Flask
- MongoDB
- HTML
- CSS
- JavaScript
